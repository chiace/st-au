# Section Blueprint

## Home Page

### 1. Hero
- Editorial headline over a premium but readable atmosphere
- Primary CTA: `Get involved`
- Secondary CTA: `Explore opportunities`
- Tertiary action: `View members`
- Premium animated centerpiece with supporting event links and proof

### 2. Signal Strip
- A lightweight moving activity band
- Used as a pacing break between hero and core content

### 3. Mission
- Two-column introduction
- Left: section title and framing
- Right: audience cards plus one strong summary sentence

### 4. Impact
- Five proof cards in a responsive grid
- Animated counters on scroll

### 5. Support
- Intro row
- Gradient editorial panel on the left
- Six support pillars on the right with icon, summary, and bullet points

### 6. Events
- Featured Luma event card
- Three secondary event cards
- Visible `View all events on Luma` CTA

### 7. Meetup Story
- Narrative image-led sequence
- Each row alternates text and media

### 8. Members
- Three featured contributor cards
- Directory CTA below

### 9. Ecosystem
- Partner grid with subtle hover treatment
- Institutional engagement panel

### 10. Community
- X embed / fallback rail
- Three utility cards
- Social proof notes
- Testimonials row
- One nerdy builder CTA banner

### 11. FAQ
- Accordion list
- One question open by default

### 12. Final Join CTA
- Strong headline
- Onboarding button
- Members button
- Channel links: Telegram, Discord, X

### 13. Footer
- Brand block
- Navigation block
- External links block
